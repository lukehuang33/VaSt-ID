Welcome to VaSt-ID, the Variable Star Classifier!

In order to use this classifier, you must import a file with three columns of values in this order: date, magnitude, error. The date must be in Julian Days. The three columns must be separated by a single space, and no non-numbers should be included.

Example input:
5346.92962 16.780 0.009
5347.91038 16.784 0.009
5358.90048 16.820 0.009
5364.91330 16.745 0.008
5376.93533 16.676 0.008
5378.93673 16.740 0.008
5380.90994 16.811 0.009
5381.91905 16.730 0.008

where the first column is the date, the second column is the magnitude, and the third column is the error.
